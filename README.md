# Juntang - Practical Tools

A lightweight personal portfolio for showcasing practical tools, Android apps, and developer utilities.

🔗 **Live Website**: [yjuntang.github.io](https://yjuntang.github.io)

---

## Design Philosophy & Tech Stack
- **Apple-Inspired Minimalism**: A design focused on content and usability, employing rich typography, muted slate gradients, and plenty of balanced whitespace.
- **Progressive Blur**: Premium glassmorphic blur transitions on scroll at the top and bottom of the viewport using native CSS masking.
- **Zero Dependencies**: Hand-crafted with pure, semantic HTML5, vanilla CSS custom properties, and lightweight vanilla Javascript.
- **Security & Performance**: Implements a strict Content Security Policy (CSP), clean metadata, and responsive layout styling optimized for speed and SEO.

## What's Inside
- `index.html` - Main page content, project cards, metadata, and contact links.
- `css/style.css` - Design system, responsive layout, themes, and animations.
- `js/main.js` - Theme toggle, scroll effects, and small interaction logic.
- `assets/` - Project icons and social preview image.
- `robots.txt` / `sitemap.xml` - Search engine discovery files.

## Local Development
Serve the site locally with any static file server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
